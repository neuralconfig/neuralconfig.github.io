Why This Combination Works

Inter is designed by Rasmus Andersson specifically for user interfaces and computer screens. It has excellent legibility at small sizes, carefully adjusted metrics, and looks professional without being boring. The tall x-height and open apertures make it extremely readable.

This combination is ideal if you want to:

  • Maximize readability for longer text content
  • Keep your familiar IBM Plex Mono for code
  • Achieve a more polished, professional appearance
  • Use a battle-tested font used by GitHub, Mozilla, and others
  • Minimal change with maximum impact

Inter's Special Features

Inter includes tabular figures (for aligned numbers in tables), contextual alternates that improve specific letter combinations, and carefully designed metrics that work beautifully across all sizes from 10px to 72px.
